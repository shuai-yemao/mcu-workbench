## 写作要求

- 代码片段必须来自项目，并标注 `文件路径:行号`；
- Mermaid 图必须反映真实调用链、数据流或时序，不能只画抽象框；
- 内部笔记使用 `[[笔记文件名]]`，不写机器绝对路径；
- 事实、用户确认、推断和未验证项分开标注；
- 笔记应保留用户回答，不把 AI 纠正伪装成用户已掌握。
